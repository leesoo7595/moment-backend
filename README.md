# Moment-backend

다양한 카테고리를 활용하여 자신만의 프라이베이트한 지도 다이어리를 만들 수 있는 어플 입니다.

you can make a special diary only for yourself.

- NodeJS
- Express
- AWS Elastic Beanstalk
- RDS Postgresql
